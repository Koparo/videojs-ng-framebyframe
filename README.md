## Installation

Add this line to your application's Gemfile:

```ruby
gem 'videojs-ng-framebyframe'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install videojs-ng-framebyframe

## Usage

Include the plugin in your `application.js`:

	//= video-framebyframe
